# Statement of Work (SOW)

## Scope

### Included

- LinkedIn safe scraping (read-only)
- Remote job API integration
- AI-based job matching
- Email auto-apply (email-only)
- Daily job summary email
- Application tracking dashboard
- PostgreSQL data storage
- Scheduled daily automation

---

### Excluded (Phase 1)

- LinkedIn auto-apply automation
- Resume rewriting per job
- Login-based scraping
- Multi-user SaaS

---

## Deliverables

### Phase 1 (MVP)
- Job collection
- AI scoring
- Email summary

### Phase 2
- Auto email apply
- Application tracking

### Phase 3
- Dashboard analytics
- SaaS-ready architecture
